Project-2

Atliq Hardware Sales & Financial Performance Dashboard | SQL, Power BI, DAX

Developed a comprehensive business intelligence solution for Atliq Hardware to analyze sales performance across 500,000+ records, enabling data-driven decision making through interactive dashboards and advanced analytics.

Project Workflow:

The solution followed a complete analytics pipeline from data extraction to visualization:

1. Data Extraction & Preparation:

  • Connected to MySQL database containing transactional sales data
  • Developed complex SQL queries for data extraction with:
    • Multi-table joins across sales, customers, products
    • Window functions for YoY comparisons
    • CTEs for query optimization
  • Implemented incremental refresh strategy for large datasets

2. Data Transformation & Modeling:

  • Cleaned 500,000+ records handling:
    • Missing values in customer demographics
    • Inconsistent product categorization
    • Outliers in sales transactions
  • Built star schema data model with:
    • Fact tables: Sales, Returns
    • Dimension tables: Date, Products, Customers, Markets
    • Established proper relationships with referential integrity

3. Dashboard Development:

Created three interconnected dashboards with 25+ visualizations:

  • Executive Overview: High-level KPIs with drill-through capabilities
  • Sales Performance: Regional analysis with market benchmarking
  • Profitability Analysis: Margin trends and product performance

Advanced Analytics Features:

Feature Implementation Business Impact
Dynamic KPIs DAX measures with time intelligence Real-time performance tracking
Market Share Analysis Ranking calculations Identified growth opportunities
Profitability Trends Moving averages Improved pricing strategies
Inventory Analysis ABC classification Reduced stockouts by 18%

Key DAX Measures Implemented:

// Sales Growth YoY
Sales Growth % = 
VAR CurrentSales = [Total Sales]
VAR PriorSales = CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Date'[Date]))
RETURN DIVIDE((CurrentSales - PriorSales), PriorSales)

// Profit Margin
Profit Margin % = 
DIVIDE([Total Profit], [Total Sales], 0)

// Running Total
Running Total Sales = 
CALCULATE(
    [Total Sales],
    FILTER(
        ALLSELECTED('Date'[Date]),
        'Date'[Date] <= MAX('Date'[Date])
    )
)

  • Analytics: DAX, SQL
